不定宽块状元素居中

1. 将元素放入table>td标签中，然后设置table的margin:0 auto这是因为table的宽度等于其内容的长度，即可以看做一个定宽的元素，从而进行定宽块的居中方法

2. 设置元素为inline类型，后设置其父元素```text-aline:center```

3. 父元素：position:relative;margin-left:50%;子元素：position:absolute;left:-50%;

用于分开上下div的div
